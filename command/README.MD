### 装饰模式（Decorator pattern）
 
##### 创建模式（Creational Pattern）
 
 
#### 定义
 
***Allows for the dynamic wrapping of objects in order to modify their existing responsibilities and behaviors.***
 
***动态地给一个对象添加一些额外的职责或者功能。***
